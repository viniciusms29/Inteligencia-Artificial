O código implementado foi rodado no sistema operacional Windows 10x64bits, com a versão do python 3.6.5

Para executar o programa deve-se seguir os seguintes parametros: 
		
		python3 main.py nomeArqEntrada.txt

	Existe dois arquivos para leitura do arquivo, o "entraDados.py", serve para executar os arquivos .txt entrada01, 02 e 03. Para isso deve-se alterar no codigo do qual arquivo .py ira ser importado.

	Os outros arquivos .txt contidos na pasta podem ser executados importando o "entradaDados2.py", que já é definido como padrão para ser possível executar os benchmaks de testes.

	Obs: Para facilitar a leitura da melhor sulução, é gerado um arquivo chamado "solucao.txt", cujo é mostrada a melhor solução encontrada para a instancia executada.
